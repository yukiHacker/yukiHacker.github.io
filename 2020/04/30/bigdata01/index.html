linux学习

1、[root@JD ~]#
root：默认管理员，最大权限
@：写死的字符
JD：机器名称
~：当前该用户的’家目录’
[]：格式规范
#：写死的字符

2、基础命令
(1)pwd：查看当前光标所在的目录/路径—— /root

(2)mkdir bigData：创建名为’bigData’的文件夹 创建文件后面再讲
mkdir dir1 dir2 dir3 ：以空格为分隔，产生并联的三个文件夹
mkdir dir4/dir5/dir6：报错
mkdir -p dir4/dir5/dir6 ：串 级联创建

(3)cd：切换目录（目录指的是路径） 只能进入文件夹，不能进入文件
如：cd /——– 从[root@JD ~]#变成了 [root@JD /]#
(Linux系统是从根目录开始的，根目录标识：/)
cd dir1 进入dir1文件夹
cd ../ 退上一层目录
cd ../../ 退2层
cd ../../../ 退3层
注意：
root用户的家目录 ： /root 是根目录下的1号文件夹
普通xx用户的家目录 ： /home/xx
家目录是用 ~ 表示
如何进家目录:cd /root
cd 直接回车
cd ~
cd -：回退到上一次的目录

(4)ls 显示文件夹、 文件名称
[root@JD /]# ls——–查看根目录下的文件夹

(5)ls -l ：显示文件夹/文件的额外信息 权限 用户用户组 时间 大小

(6)ls -l -a: 也显示隐藏文件夹 文件（隐藏文件夹 文件是以.开头 ）
如：
[root@ruozedata001 ~]# ls -l -a
total 40
dr-xr-x—. 6 root root 4096 Apr 15 21:51 .
dr-xr-xr-x. 17 root root 4096 Aug 8 2018 ..
-rw——-. 1 root root 3814 Apr 15 21:49 .bash_history
-rw-r–r–. 1 root root 18 Dec 29 2013 .bash_logout

(7)ls -l -h： 仅仅查看文件的大小（文件夹的大小也显示，但是不准确）
后面会学习查看文件夹的大小

(8)ls -l -r -t：按时间排序(如何快速找到哪些文件更新了?)

(9)ls -l 和 ll 等价
ll -a
ll -rt
ll -h

(10)命令帮助 help
如：ls –help 即查看关于ls的命令帮助
root@ruozedata001 /]# ls –help
Usage: ls [OPTION]… [FILE]…
解析：[]标识的 可选
… 多个参数

Usage: ls [OPTIONS]和Usage: ls [OPTION]… [FILE]… 是一样的意思
Usage: ls XXX [OPTIONS] ：xxx必须要写

List information about the FILEs (the current directory by default).
Sort entries alphabetically if none of -cftuvSUX nor –sort is specified.

Mandatory arguments to long options are mandatory for short options too.
-a, –all do not ignore entries starting with .
-A, –almost-all do not list implied . and ..
